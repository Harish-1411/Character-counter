# Character Counter

A simple and interactive **Character Counter** built using **HTML, Tailwind CSS, and JavaScript**.
This tool allows users to type text in a textarea and see the **live character count** instantly.

---

## Features

* Live character counting while typing
* Clean and modern UI using **Tailwind CSS**
* Maximum character limit support
* Color change when the limit is reached
* Responsive design

---

## Technologies Used

* **HTML5** – Page structure
* **Tailwind CSS** – Styling and layout
* **JavaScript** – Character counting functionality
